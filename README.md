## 从零开始建站：用搬瓦工 VPS 跑 WordPress 的完整步骤

拿到 VPS 之后，建站分这几个阶段走：

**阶段一：基础准备**

1. 注册域名（搬瓦工本身不提供域名服务，需要另行购买）
2. 将域名 A 记录解析到搬瓦工 VPS 的 IP 地址
3. 通过 SSH 连接 VPS（KiwiVM 面板里可以看到 IP 和 SSH 端口）

**阶段二：部署建站环境**

两条路可以走，看自己习惯：

**路线 A：LNMP 一键包（命令行）**

bash
wget https://soft.lnmp.com/lnmp/lnmp2.0.tar.gz -O lnmp2.0.tar.gz && \
tar zxf lnmp2.0.tar.gz && cd lnmp2.0 && ./install.sh lnmp


安装过程大概需要 20-40 分钟，取决于机房网速。完成后 Nginx + MySQL + PHP 环境就位。内存不足 1GB 的情况下，建议 MySQL 选 5.5 版本，PHP 建议 7.4 以上。

**路线 B：宝塔面板（图形界面，适合新手）**

宝塔面板一行命令装好，之后所有操作都在 Web 界面里完成，建站、上传文件、配置 SSL 全部可视化。门槛比 LNMP 低不少，但会多占一些内存，1GB 机器运行起来会比较紧张——建议 1.5GB 以上比较稳。

**阶段三：安装 WordPress**

1. 下载 WordPress 安装包，上传到服务器对应目录
2. 创建 MySQL 数据库和用户，记住数据库名、用户名、密码
3. 浏览器访问域名，按提示完成 WordPress 安装向导
4. 配置好伪静态规则（LNMP 环境下需要在面板里手动设置）

**阶段四：SSL 证书和安全设置**

Let's Encrypt 免费证书，LNMP 一键包自带申请命令，宝塔面板里也有一键申请入口。装完 SSL 之后记得把 HTTP 重定向到 HTTPS，这一步不做会影响 SEO。
